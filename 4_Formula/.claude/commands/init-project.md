# /init-project

Bootstrap all 7 stage folders and root config files for a new project using this template.

## Usage
```
/init-project
/init-project "My Project Name"
```

## What this command does

Sets up a new project from `delivery-pilot-template` by:

1. Confirming the project name (used in `CLAUDE.md`, `README.md`, `agents.md`)
2. Creating the 7 stage folders if they don't exist, each with a `README.md`
3. Copying `.claude/` config to root (settings, commands)
4. Running `/init` to generate a fresh `CLAUDE.md` for the new codebase
5. Creating `index.html` with both Project Menu and Debug Menu wired up
6. Initializing `6_Semblance/error.log` and `6_Semblance/fix.log`
7. Making the first commit: `chore: initialize project from delivery-pilot-template`

## Instructions for the agent

When the user runs `/init-project`:

### Step 1 — Gather inputs
- Project name (required)
- GitHub username (default: rifaterdemsahin)
- Repo name (used for GitHub Pages URL)

### Step 2 — Create folder structure
```
1_Real_Unknown/README.md   — The "Why": OKRs and problem definitions
2_Environment/README.md    — The "Context": setup guides and constraints
3_Simulation/README.md     — The "Vision": UI mockups and images
4_Formula/README.md        — The "Recipe": step-by-step guides
5_Symbols/README.md        — The "Reality": source code
6_Semblance/README.md      — The "Scars": errors, workarounds, lessons
7_Testing_Known/README.md  — The "Proof": validation and checklists
```

### Step 3 — Configure .claude at root
- Copy `4_Formula/.claude/settings.template.json` → `.claude/settings.json`
- Copy `4_Formula/.claude/commands/` → `.claude/commands/`

### Step 4 — Initialize tracking files
- Create `6_Semblance/error.log` with header template
- Create `6_Semblance/fix.log` with header template
- Create `6_Semblance/lessons_learned.md` with init entry
- Create `prompts.md` with header

### Step 5 — First commit
```
git add .
git commit -m "chore: initialize <project-name> from delivery-pilot-template"
git push -u origin main
```

- [ ] All 7 folders exist with README.md
